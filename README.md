## 구현목표
### 필수구현사항
* SignUp/SignIn:100:
* SignIn Decorator:100:
* Product List:100:
* Product Detail:100:
* Cart:100:
  
### 추가구현사항(추가 예정)
* Order
* Review
* Mypage

### 활용 라이브러리
* django-cors-headers
* bcrypt
* jwt

* 김동규 : 전체(Product List), 카테고리(Category List), 장바구니(CRUD), Filtering/Ordering/Pagination 기능 구현
